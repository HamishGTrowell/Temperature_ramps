## Dependencies

The 'requirements.txt' file lists the core Python dependencies required to run the script.
Exact package versions may vary slightly depending on platform and Python version.

The script was developed using:
- Python 3.13.5
- numpy 2.3.1
- pandas 2.3.2
- scipy 1.16.0
- matplotlib 3.10.5

## Data

The dataset provided corresponds to the data used in Chapter III of the thesis.
The data are provided for examination purposes only and should not be redistributed without permission.

## Usage

Data analysis is conducted via command-line control of cli.py.
To run on the example datasets, try:
```
python cli.py example-data_1 --showplots --model eyring
```
For information on additional commands, try:
```
python cli.py -h
```
